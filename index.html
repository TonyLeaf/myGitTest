如题： 解答： <!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>百度试题解答</title> <style> #demo{ background-color: #fff; height:100px; width:100px; border: 2px solid black; margin:30px auto; } .test2{ height: 20px; width: 20px; border: 2px solid black; margin-top: 20px; margin-left: 90px; border-bottom: 0; border-left: 0; transform: rotate(45deg); background: #fff; z-index: 1; } </style> </head> <body> <div id="demo"> <div class="test2"></div> </div> </body> </html> 效果图： 优点：效果完美
缺点：不兼容IE浏览器
-------------------------------------------------------------------------------------- 以下是公司的代码小王子教我的其他思路
<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <title>input练习</title> <style type="text/css"> .s{ width:0; height: 0; border-color: transparent #f40; border-width: 20px; border-style: solid; } </style> </head> <body> <div class="s"></div> </body> </html> 原理： 右边突出来的那一块，直接用CSS构造出一个三角形，构造三角形的方法是， div本身长宽设为0，div的边框设大，然后把其他三边设为透明，留下其中一边。


优点： 能够直接构造出一个三角形，不需要通过旋转div，去其两边，Z轴正向移动1的方法来做。
缺点： 无法给这个三角形本身设置边框。
